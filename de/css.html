CSS-Grundlagen

Um ein Element in HTML zu gestalten, wähle zwischen zwei Optionen:
Option 1:
Füge das Element <style> unter dem Code hinzu, den du gestalten möchtest.
Option 2:
Erstelle eine neue Datei namens "style.css" und beginne dort mit deinem Code. In deiner Datei "style.css" kannst du ein HTML-Element auswählen und ihm Stil geben.
Zum Beispiel, um die Hintergrundfarbe der Seite zu ändern, schreibe:
body {
  background-color: lightblue;
}

Um eine Überschrift wie <h1> zu gestalten, schreibe:
h1 {
  color: darkgreen;
  font-family: Arial, sans-serif;
  text-align: center;
}

Um einen Absatz zu gestalten, verwende:
p {
  font-size: 18px;
  color: black;
}

Du kannst auch einen eigenen Klassennamen vergeben und ihn so gestalten:
.my-box {
  border: 2px solid red;
  padding: 10px;
  background-color: yellow;
}
