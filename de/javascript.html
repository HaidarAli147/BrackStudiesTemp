JavaScript-Grundlagen

JavaScript wird verwendet, um deine Website interaktiv zu machen.
Du kannst JavaScript auf zwei Arten schreiben:
Option 1: Direkt in der HTML-Datei mit dem <script>-Tag.
Option 2: In einer separaten Datei namens "script.js" und diese in HTML verlinken. Beispiel für JavaScript in HTML:
<script>
  alert("Willkommen bei Brack Study!");
</script>

Beispiel in "script.js":
function greet() {
  alert("Hallo aus externer Datei!");
}

Um die Funktion beim Klick auf einen Button auszuführen:
<button onclick="greet()">Klick mich</button>
