أساسيات JavaScript

تُستخدم JavaScript لجعل موقعك الإلكتروني تفاعليًا.
يمكنك كتابة JavaScript بطريقتين:
الخيار الأول: داخل ملف HTML باستخدام الوسم <script>.
الخيار الثاني: في ملف منفصل يسمى "script.js" وربطه بصفحة HTML. مثال على JavaScript داخل HTML:
<script>
  alert("مرحبًا بك في Brack Study!");
</script>

مثال في "script.js":
function greet() {
  alert("مرحبًا من الملف الخارجي!");
}

لتشغيل هذه الوظيفة عند الضغط على زر:
<button onclick="greet()">اضغط هنا</button>
