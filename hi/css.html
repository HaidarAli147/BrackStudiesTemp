CSS की मूल बातें

HTML में किसी तत्व को स्टाइल करने के लिए दो विकल्प होते हैं:
विकल्प 1:
उस कोड के नीचे <style> टैग जोड़ें जिसे आप स्टाइल करना चाहते हैं।
विकल्प 2:
एक नई फ़ाइल बनाएं जिसका नाम हो "style.css" और उसमें अपना कोड लिखें। "style.css" फ़ाइल में आप किसी HTML तत्व को चुनकर उसे स्टाइल दे सकते हैं।
उदाहरण के लिए, पूरे पेज की बैकग्राउंड रंग बदलने के लिए लिखें:
body {
  background-color: lightblue;
}

<h1> जैसे शीर्षक को स्टाइल करने के लिए लिखें:
h1 {
  color: darkgreen;
  font-family: Arial, sans-serif;
  text-align: center;
}

पैराग्राफ को स्टाइल करने के लिए लिखें:
p {
  font-size: 18px;
  color: black;
}

आप किसी भी तत्व को अपनी क्लास देकर इस तरह स्टाइल कर सकते हैं:
.my-box {
  border: 2px solid red;
  padding: 10px;
  background-color: yellow;
}
