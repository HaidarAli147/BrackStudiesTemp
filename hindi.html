होम पाइथन

पाइथन प्रोजेक्ट्स पेज में आपका स्वागत है

हम कोडिंग सामग्री बनाते हैं ताकि यह दिखा सकें कि इलेक्ट्रॉनिक्स के पीछे क्या चल रहा है...

Python एक कोडिंग भाषा है जिसका उपयोग प्रोग्राम (ऐप्स) बनाने के लिए किया जाता है। कुछ प्रिंट करने के लिए:

print('hello')


इनपुट लेने के दो तरीके हैं:

पहला तरीका:

input("आपका नाम क्या है?")

यह तरीका if स्टेटमेंट में उपयोग नहीं किया जा सकता

दूसरा तरीका:

i = input("आपका नाम क्या है?")

यह तरीका if स्टेटमेंट में उपयोग किया जा सकता है

अगर हम if स्टेटमेंट की बात करें, तो इसे इस तरह लिखा जाता है:

i = int(input("आपकी उम्र क्या है? "))
if i >= 18:
    print("आप हमारी कंपनी की शर्तों को पूरा करते हैं!")
else:
    print("आप हमारी कंपनी की शर्तों को पूरा नहीं करते, इसलिए आप खाता नहीं बना सकते!")

डेटा प्रकार

टेक्स्ट वैल्यू (string या str) लिखने के लिए:
name = "Ali"
age = "15"
print(name)
print(age)
(बराबर चिन्ह से पहले जो नाम लिखा जाता है उसे वेरिएबल कहते हैं। आप इसे कोई भी नाम दे सकते हैं जब तक आपने वही नाम पहले इस्तेमाल न किया हो।)
is_student = "True"
पूर्णांक (Integer) लिखने के लिए:
number_of_students = 10
speed = -120
print(number_of_students)
print(speed)
दशमलव संख्या (float) लिखने के लिए:
GPA = 3.5
bill = round(15.329, 2)
print(bill)
print(GPA)
Boolean (True या False) लिखने के लिए:
is_player = True
#या is_footballer = False
print(is_player)
print(is_footballer)
(**हैश चिन्ह (#)** का मतलब है कि उसके बाद का कोड एक टिप्पणी है जिसे प्रोग्राम नजरअंदाज करता है।)
अगर आप उसके मान को बदलना नहीं चाहते तो लिखें:
is_player = True
#या is_footballer = False
print(not is_player)
print(not is_footballer)

गणितीय समीकरण

जोड़ करने के लिए:
print(2+2)
#सावधान: यहाँ बराबर चिन्ह का उपयोग न करें!
घटाव करने के लिए:
print(10-6)
गुणा करने के लिए:
print(9*9)
#सावधान: 'x' की जगह * का उपयोग करें
भाग करने के लिए:
print(28/4)
#सिर्फ स्लैश चिन्ह का उपयोग करें
मॉड्यूलो (बचा हुआ भाग) के लिए:
print(24%7)
तुलना ऑपरेटर (==, !=, >, <, >=, <=) के लिए:
print(1 == 1)
print(1 != 2)
print(1 > 2)
print(1 < 2)
print(1 >= 2)
print(1 <= 2)
("==" का मतलब बराबर, "!=" का मतलब बराबर नहीं, ">" का मतलब बड़ा, "<" का मतलब छोटा, ">=" का मतलब बराबर या बड़ा, "<=" का मतलब बराबर या छोटा।)
