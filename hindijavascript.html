JavaScript की मूल बातें

JavaScript का उपयोग आपकी वेबसाइट को इंटरएक्टिव बनाने के लिए किया जाता है।
आप JavaScript दो तरीकों से लिख सकते हैं:
विकल्प 1: HTML फ़ाइल में <script> टैग के अंदर।
विकल्प 2: एक अलग फ़ाइल "script.js" बनाकर और उसे HTML में लिंक करके। HTML में JavaScript का उदाहरण:
<script>
  alert("Brack Study में आपका स्वागत है!");
</script>

"script.js" में उदाहरण:
function greet() {
  alert("बाहरी फ़ाइल से नमस्ते!");
}

इस फ़ंक्शन को बटन क्लिक पर चलाने के लिए:
<button onclick="greet()">यहाँ क्लिक करें</button>
