HTML, which stands for HyperText Markup Language, is the fundamental building block of the web. It's not a programming language, but rather a markup language used to structure the content you see on the internet. Think of HTML as the skeleton of a webpage; it defines all the different partsâ€”like the heading, paragraphs, images, and linksâ€”so your web browser knows how to display them. Every webpage you've ever visited uses HTML.

The Core Concept: Elements and Tags

At the heart of HTML are elements. An HTML element is everything from the start tag to the end tag, including the content in between.

Elements are created using tags, which are keywords enclosed in angle brackets, like <p>. Most elements have a start tag (or opening tag) and an end tag (or closing tag). The closing tag includes a forward slash: </p>.

Example: <p>This is a paragraph.</p>

Here, <p> is the opening tag, </p> is the closing tag, and "This is a paragraph" is the content. Together, they form the paragraph element.

Some elements are self-closing (or void elements) and don't require a closing tag because they don't wrap around content. A common example is the image element: <img src="image.jpg" alt="Description">.

Structure of an HTML Document

Every well-formed HTML document follows a basic structure. It always begins with a document type declaration and includes four essential elements:

  1. <!DOCTYPE html>: This declaration defines that the document is an HTML5 document. It must be the very first thing in your code.
  2. <html>: This is the root element and wraps all the other content on the entire page. It often includes an attribute to declare the language of the document, such as <html lang="en">.
  3. <head>: This section contains metadata about the HTML document. This is information not displayed directly on the webpage but is crucial for the browser and search engines. It includes things like the page title, character set, and links to external files (like CSS stylesheets).
    • <title>: Sets the title that appears in the browser tab.
  4. <body>: This is where all the visible content of your page goes. Everything a user seesâ€”text, images, links, videosâ€”is nested inside the <body> element.

Key HTML Elements You'll Use

To create compelling content, you'll need to use various HTML elements:

  • Headings: Used to structure your text, they range from <h1> (most important/largest) down to <h6> (least important/smallest). 
    <h1>Welcome to my site</h1>
  • Paragraphs: Used for blocks of text. 
    <p>Here is some important information.</p>
  • Links (Anchors): Used to connect to other pages or resources. They use the href attribute to specify the destination URL. 
    <a href="https://www.example.com">Visit Example</a>
  • Images: Used to embed pictures. They use the src attribute for the image file path and the alt attribute for a text description (important for accessibility and SEO). 
    <img src="photo.jpg" alt="A photo of a landscape">
  • Lists: Used to organize items, either unordered (bullet points, <ul> and <li>) or ordered (numbered, <ol> and <li>).
