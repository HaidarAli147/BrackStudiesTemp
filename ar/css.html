أساسيات CSS

لتنسيق عنصر في HTML، اختر بين خيارين:
الخيار الأول:
أضف العنصر <style> تحت الكود الذي تريد تنسيقه.
الخيار الثاني:
أنشئ ملفًا جديدًا باسم "style.css" وابدأ كتابة الكود بداخله. داخل ملف "style.css"، يمكنك اختيار عنصر HTML وتطبيق التنسيق عليه.
على سبيل المثال، لتغيير لون خلفية الصفحة بالكامل، اكتب:
body {
  background-color: lightblue;
}

لتنسيق عنوان مثل <h1>، اكتب:
h1 {
  color: darkgreen;
  font-family: Arial, sans-serif;
  text-align: center;
}

لتنسيق فقرة، استخدم:
p {
  font-size: 18px;
  color: black;
}

يمكنك أيضًا إعطاء اسم فئة خاص بك لأي عنصر وتنسيقه بهذا الشكل:
.my-box {
  border: 2px solid red;
  padding: 10px;
  background-color: yellow;
}
