Willkommen auf der Python-Projektseite

Wir erstellen Programmierinhalte, um zu zeigen, was hinter den Kulissen der Elektronik passiert...

Python ist eine Programmiersprache, mit der man Programme (Apps) erstellen kann. Um etwas auszugeben:

print('hello')


Um eine Eingabe zu machen, gibt es zwei Möglichkeiten:

Möglichkeit Nummer eins:

input("Wie heißt du?")

Diese Methode kann nicht in einer if-Anweisung verwendet werden

Möglichkeit Nummer zwei:

i = input("Wie heißt du?")

Diese Methode kann in einer if-Anweisung verwendet werden

Zum Thema if-Anweisungen – so schreibt man eine:

i = int(input("Wie alt bist du? "))
if i >= 18:
    print("Du erfüllst die Bedingungen unseres Unternehmens!")
else:
    print("Du erfüllst die Bedingungen unseres Unternehmens nicht, daher kannst du kein Konto erstellen!")

Datentypen

Um einen String (str) zu schreiben, also Textwerte:
name = "Ali"
age = "15"
print(name)
print(age)
(Der Name vor dem Gleichheitszeichen ist eine Variable. Du kannst sie beliebig benennen, solange du den Namen nicht bereits im selben Bereich verwendet hast.)
is_student = "True"
Um Ganzzahlen (positive und negative Zahlen) zu schreiben:
number_of_students = 10
speed = -120
print(number_of_students)
print(speed)
So schreibt man eine Kommazahl (float):
GPA = 3.5
bill = round(15.329, 2)
print(bill)
print(GPA)
Um einen Boolean (True oder False) zu schreiben:
is_player = True
#oder is_footballer = False
print(is_player)
print(is_footballer)
(Das **Rautezeichen (#)** bedeutet, dass der folgende Code ein Kommentar ist. Der Interpreter ignoriert ihn und er wird beim Ausführen der App nicht angezeigt.)
Wenn du den Wert nicht ändern möchtest (True oder False), schreibe:
is_player = True
#oder is_footballer = False
print(not is_player)
print(not is_footballer)

Mathematische Gleichungen

Für Addition:
print(2+2)
#Achtung: Kein Gleichheitszeichen verwenden!
Für Subtraktion:
print(10-6)
Für Multiplikation:
print(9*9)
#Achtung: Der Buchstabe 'x' funktioniert nicht – verwende das Sternchen (*)
Für Division:
print(28/4)
#Nur den Schrägstrich verwenden
Für Modulo (Restwert):
print(24%7)
Für Vergleichsoperatoren (==, !=, >, <, >=, <=):
print(1 == 1)
print(1 != 2)
print(1 > 2)
print(1 < 2)
print(1 >= 2)
print(1 <= 2)
("==" bedeutet gleich, "!=" ungleich, ">" größer als, "<" kleiner als, ">=" größer oder gleich, "<=" kleiner oder gleich.)
